## 注意
yjj数据库的编译需要依赖boost库。为避免自己编译库所带来的不必要的麻烦，可直接使用包管理器下载：
```
    sudo apt-get install libboost-all-dev
```
all是必须的，否则仍然会报错。之后可以完全按照文档所述编译数据库。
